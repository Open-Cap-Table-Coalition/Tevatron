### Dividend Interest Type

**Description:** _- SIMPLE_INTEREST: Interest is calculated using simple interest
 - INTEREST_PERIOD_DAILY: Interest compounds daily
 - INTEREST_PERIOD_WEEKLY: Interest compounds weekly
 - INTEREST_PERIOD_MONTHLY: Interest compounds monthly
 - INTEREST_PERIOD_QUARTERLY: Interest compounds quarterly
 - INTEREST_PERIOD_SEMI_ANNUAL: Interest compounds semi-annually
 - INTEREST_PERIOD_ANNUAL: Interest compounds annually_
