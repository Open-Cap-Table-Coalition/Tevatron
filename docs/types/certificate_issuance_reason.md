### Certificate Issuance Reason

**Description:** _The reason a certificate was created. Indicates what originated the certificate.

 - CERTIFICATE_ISSUANCE_REASON_ISSUED_FROM_SHARE_RESERVE: Certificate was issued directly from the share reserve.
 - CERTIFICATE_ISSUANCE_REASON_OPTION_EXERCISED: Certificate was created as a result of an option exercise.
 - CERTIFICATE_ISSUANCE_REASON_RSU_SETTLED: Certificate was created as a result of an RSU settlement.
 - CERTIFICATE_ISSUANCE_REASON_DEBT_CONVERTED: Certificate was created as a result of a convertible note conversion.
 - CERTIFICATE_ISSUANCE_REASON_WARRANT_EXERCISED: Certificate was created as a result of a warrant exercise.
 - CERTIFICATE_ISSUANCE_REASON_SHARE_CLASS_CONVERTED: Certificate was created as a result of a share class conversion.
 - CERTIFICATE_ISSUANCE_REASON_TRANSFERRED: Certificate was created as a result of a transfer._
