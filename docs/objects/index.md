# API Objects

Primary API objects that have associated endpoints in the Carta Issuer API. These are the main resources you can create, read, update, or delete through the API.

## Available Objects

- [Benchmark Job](benchmark_job.md) - Compensation benchmark job information
- [Benchmarks](benchmarks.md) - Compensation benchmark data
- [Certificate](certificate.md) - Stock certificates
- [Convertible Note](convertible_note.md) - Convertible notes issued to investors
- [Corporation](corporation.md) - Corporation information
- [Draft Securities Service Create Draft Option Grant Body](draft_securities_service_create_draft_option_grant_body.md) - Request body for creating draft option grants
- [Exercise Service Update Money Movement Body](exercise_service_update_money_movement_body.md) - Request body for updating exercise money movements
- [Exercise Service Update Tax Withholding Body](exercise_service_update_tax_withholding_body.md) - Request body for updating exercise tax withholding
- [Fair Market Value](fair_market_value.md) - Fair market valuations for share classes
- [Interest](interest.md) - Interests for an issuer
- [Issuer](issuer.md) - Issuer information
- [Option Exercise](option_exercise.md) - Option exercise transactions
- [Option Grant](option_grant.md) - Stock option grants
- [Point Of Contact](point_of_contact.md) - Points of contact for an issuer
- [Publicapiissuers Stakeholder](publicapiissuers_stakeholder.md) - Stakeholder information
- [Restricted Stock Award](restricted_stock_award.md) - Restricted stock awards (RSAs)
- [Restricted Stock Unit](restricted_stock_unit.md) - Restricted stock units (RSUs)
- [Share Class](share_class.md) - Share classes and their properties
- [Vesting Schedule Template](vesting_schedule_template.md) - Templates for vesting schedules
